# 任务管理系统

一个现代化的任务管理系统，包含管理员后台和员工工作台两套界面，支持智谱AI集成。

## 🎨 界面风格

- **浅色主题**：清新明亮的浅色设计
- **现代化UI**：流畅动画、渐变配色、卡片式布局
- **响应式设计**：适配不同屏幕尺寸

## ✨ 功能特性

### 管理员后台 (admin.html)
- 📊 **数据看板**：实时查看任务完成率、员工负荷、词条健康度等核心指标
- 💬 **AI任务生成**：通过智谱AI对话智能拆解大任务，自动匹配员工技能
- 📋 **任务管理**：看板视图和层级视图管理所有任务
- 👥 **员工管理**：查看员工技能、贡献度、任务负荷等信息
- ⚙️ **系统设置**：配置智谱AI API、任务规则、界面主题等

### 员工工作台 (employee.html)
- 📋 **我的任务**：看板视图展示待处理、进行中、待验收、已完成任务
- 👤 **个人资料**：查看个人技能、贡献度、等级徽章
- 📎 **任务提交**：上传文件提交任务，支持拖拽上传
- 🤖 **AI评分**：自动评估词条复现率和内容质量
- 🔔 **通知中心**：接收任务通知、验收结果等消息
- 🔄 **账户切换**：快速切换不同员工账户

## 🚀 快速开始

### 1. 打开登录页面
直接在浏览器中打开 `index.html`

### 2. 登录方式

#### 管理员登录
- 点击"管理员登录"标签
- 快速登录：点击"管理员"按钮

#### 员工登录
- 点击"员工登录"标签
- 快速登录：点击"林诗雨"、"陈星辰"、"苏晓月"或"江逸凡"

### 3. 配置智谱AI（可选）

如果你想使用真实的AI功能：

1. 访问 [智谱AI开放平台](https://open.bigmodel.cn/)
2. 注册账号并获取免费的API Key
3. 登录管理员后台，进入"系统设置"
4. 输入API Key并保存
5. 点击"测试连接"验证配置

**推荐模型**：GLM-4-Flash（免费且快速）

### 4. 体验功能

#### 管理员后台
1. **查看数据看板**：查看各项统计指标和图表
2. **AI生成任务**：
   - 点击顶部"💬"按钮或左侧"AI任务生成"
   - 输入任务需求，例如：
     ```
     Q3提升'无人驾驶'词条覆盖
     目标：知乎10篇+论坛20篇
     核心词条：ULL/远大/5ms
     ```
   - AI会自动拆解任务并推荐员工
   - 确认后创建任务
3. **管理任务**：在看板视图中查看和管理所有任务
4. **管理员工**：查看员工信息、技能标签和工作负荷
5. **系统设置**：配置AI、任务规则等

#### 员工工作台
1. **查看任务**：在看板视图中查看自己的任务
2. **提交任务**：
   - 点击任务卡片打开详情
   - 上传文件（支持拖拽）
   - 点击"提交任务"
   - 查看AI评分结果
3. **切换账户**：
   - 点击右上角用户头像
   - 选择其他员工账户
   - 体验不同员工的任务视图
4. **查看个人资料**：查看贡献度、等级徽章等信息
5. **通知中心**：查看任务通知和系统消息

## 🤖 智谱AI集成说明

### 获取API Key

1. 访问 https://open.bigmodel.cn/
2. 注册并登录账号
3. 进入"API Keys"页面
4. 创建新的API Key（免费额度足够测试使用）

### 配置方法

**方法一：通过界面配置（推荐）**
1. 登录管理员后台
2. 进入"系统设置"
3. 输入API Key
4. 选择模型（推荐GLM-4-Flash）
5. 点击"保存配置"
6. 点击"测试连接"验证

**方法二：直接修改代码**
打开 `admin-data.js`，修改第一行：
```javascript
const ZHIPU_API_KEY = '你的API_Key';
```

### AI功能说明

- **任务拆解**：AI会根据你的描述自动拆解大任务为多个子任务
- **员工匹配**：根据员工技能自动推荐合适的人选
- **智能建议**：提供任务优先级、截止时间、贡献度等建议
- **Fallback机制**：如果API调用失败，会自动使用预设回复

### 费用说明

## ⚠️ 注意事项

- 这是一个前端演示系统，所有数据都是模拟的
- 刷新页面后某些操作会重置（如任务状态变更）
- 文件上传功能仅模拟，不会真正上传到服务器
- AI评分结果是随机生成的演示效果（除非配置了真实API）
- 智谱AI API需要网络连接，请确保网络畅通

## 🔧 开发说明

如需二次开发或集成后端：

1. **数据接口**：将 `*-data.js` 中的模拟数据替换为 API 调用
2. **文件上传**：在 `employee.js` 的 `submitTask` 函数中添加真实的文件上传逻辑
3. **AI集成**：已集成智谱AI，可替换为其他AI服务
4. **状态管理**：考虑使用 Vue/React 等框架进行状态管理
5. **主题切换**：可在 `styles.css` 中添加深色主题变量

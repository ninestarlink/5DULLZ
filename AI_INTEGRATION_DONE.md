## 🚀 使用方法

### 1. 访问AI任务生成
1. 用管理员登录
2. 点击侧边栏"💬 AI任务生成"
3. 或点击右上角的💬按钮

### 2. 生成任务
1. 在输入框描述需求，例如：
   ```
   为新产品设计一套完整的营销方案，包括市场调研、内容创作、社交媒体推广等
   ```
2. 点击"发送"或按 `Ctrl+Enter`
3. AI会自动生成详细的任务列表

### 3. 编辑和创建
1. 查看生成的任务列表
2. 可以修改标题、描述、优先级、分配人员
3. 点击"确认创建所有任务"
4. 任务会保存到数据库
5. 员工立即可以看到
